# NAS油条Linux工具箱 - 完整功能说明

## 脚本概述

**NAS油条Linux工具箱**是一个功能强大的Linux系统管理脚本，由NAS油条开发,该脚本集成了大量实用的Linux系统工具和功能，为用户提供一站式的系统管理和维护解决方案。


 **一键安装命令** 
```bash
bash -c "$(curl -L https://raw.gitcode.com/nasyt/nasyt-linux-tool/raw/master/nasyt_install.sh)"
```

## 🚀 使用方法

### 安装脚本
```bash
# 直接运行脚本（首次运行）
bash nasyt.sh

# 或使用在线安装
curl -fsSL https://gitee.com/nasyt/nasyt-linux-tool/raw/master/nasyt.sh | bash
```

### 基本使用
```bash
# 启动脚本
nasyt

# 快捷命令参数
nasyt -g          # 快捷更新脚本
nasyt -u          # 快捷换软件源
nasyt -t          # 快捷进入tmux管理
nasyt -s          # 直接进入菜单部分
nasyt -v          # 显示脚本版本
nasyt -h          # 显示帮助信息
```

### 脚本安装
```bash
# 安装后可直接使用nasyt命令
1) 选择"Linux工具箱 (安装)"
2) 或手动安装到系统路径
```
